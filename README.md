# Smart Health Checker (AI Expert System)

## 🔍 Description
This is a simple AI-based health checker using rule-based logic.

## ⚙️ Features
- Symptom checker (Fever, Cough, Headache)
- Instant result using IF-THEN rules
- Simple and clean UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How to Run
Open index.html in browser
